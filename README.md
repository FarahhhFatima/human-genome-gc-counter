# Human Genome GC Content Counter

## Overview
This project calculates the GC content of a human DNA sequence from a FASTA file.

## Why GC Content?
GC content is an important genomic feature used in:
- Genome characterization
- Gene prediction
- Comparative genomics

## Input
- FASTA file containing a DNA sequence (example: human chromosome)

## Output
- Sequence length
- GC percentage

## How to Run
```bash
python gc_counter.py
